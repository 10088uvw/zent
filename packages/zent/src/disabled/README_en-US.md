## Disabled

Child Zent components will be disabled. The `disabled` property of component itself has a higher priority.

### API

| Property | Description | Type | Required | Default | Alternative |
| -------- | ----------- | ---- | -------- | ------- | ----------- |
| value    |             | bool | No       | true    |             |
