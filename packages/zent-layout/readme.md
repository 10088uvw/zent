# @youzan/zent-layout

24栅格布局系统

[![version][version-image]][download-url]
[![download][download-image]][download-url]

## Row API

| 属性         | 说明            | 类型     | 默认值   |
| ----------- | -------------- | -------- | -------- |
| className   | 额外的样式名      | string   |          |
| prefix      | UI 前缀          | string   | zent     |

## Col API

| 属性         | 说明                 | 类型       | 默认值  |
| ----------- | ------------------- | -------- | -------- |
| span        | col所占的栅格数        | number   |          |
| offset      | col左偏移的栅格数       | number   |          |
| className   | 额外的样式名           | string   |          |
| prefix      | UI 前缀               | string   | zent     |
